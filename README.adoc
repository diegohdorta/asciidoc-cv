== Building Process

=== Software Requirements

. Install the essential packages:
+
[source,console]
----
~# apt install ruby-bundler
~# bundle install
----

=== Generating CV

[source,console]
----
~$ git clone https://github.com/diegohdorta/asciidoc-cv.git
~$ cd asciidoc-cv/
~asciidoc-cv$ ./asciidoc-cv-builder
----
